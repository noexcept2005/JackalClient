## 模块概览

所有模块被分为 10 大类别：
Combat（战斗）、Window（窗口）、Process（进程）、Registry（注册表）、File（文件）、Control（控制）、Render（渲染）、Web（网络）、Misc（杂项）、Driver（驱动）


一些不容错过的优秀模块：
- **AntiCapture（反捕获）：** 让客户端窗口在录屏软件中不可见。
- **ClearPin（清除PIN）：** 清除 Windows PIN。
- **Gatekeeper（看门人）：** 拦截部分 Windows 登录事件，例如 Ctrl + Alt + Delete、关机、重启、注销等。
- **TrustedInstaller（受信任安装者）：** 获取 TrustedInstaller 权限。
- **WindowHighlight（窗口高亮）：** 为当前激活窗口绘制边框，并在其下方显示暗色背景。
- **OpacityChanger（透明度调节）：** 允许通过鼠标滚轮调整窗口透明度。
- **ConsoleESP（控制台透视）：** 监控控制台变化，也可实时输出其他控制台内容。
- **AntiPopups（反弹窗）：** 帮助你摆脱弹窗。
- **DialogEscape（对话框逃逸）：** 自动关闭消息框。
- **PrivateWindow（隐私窗口）：** 让某些私密窗口（如 Edge、QQ、微信）不被录屏软件捕获。
- **ProcessManager（进程管理器）：** 内置的进程管理工具。
- **CompileESP（编译透视）：** 监控 C / C++ 编译的开始与结束。
- **SecurityESP（安全透视）：** 显示杀毒软件和恶意进程的 HUD 信息。
- **ProcessESP（进程透视）：** 显示进程创建和退出并进行通知。
- **Injector（注入器）：** 简单的注入工具，支持基础的反射注入。
- **FileManager（文件管理器）：** 内置的文件管理工具。
- **UpdateDisabler（更新禁用器）：** 自动延迟 Windows 更新。
- **AutoClicker（连点器）：** 帮助你自动点击。
- **MouseESP（鼠标透视）：** 绘制多种美观的鼠标特效。
- **ColorPicker（取色器）：** 显示鼠标指向位置的颜色信息。
- **Crosshair（准星线）：** 绘制准星。
- **SnapTap（急停键）：** 简单的 Snap - Tap 模块。
- **Keystrokes（按键显示）：** 显示游戏中的 AWSD 按键操作。
- **Keystrokes2（按键显示II）：** 以单行或多行动态显示按键。
- **KeySound（按键音效）：** 高度可自定义的按键音模块，支持配置文件。
- **ClipboardESP（剪贴板透视）：** 显示剪贴板中的文本或位图信息（你刚复制的内容）。
- **AutoL（自动嘲讽）：** 在某些 Minecraft 场景下自动发送 * **文本（不涉及注入，依赖游戏日志）。
- **ClipboardTweaks（剪贴板功能）：** 处理复制文本，包含高强度混淆方法。
- **Typewriter（打字机）：** 以极快速度自动输入指定内容。
- **Shell（命令行）：** 客户端命令行界面，拥有丰富的命令系统（详见 COMMANDS.txt）。
- **Arraylist（模块列表）：** 类似 Minecraft 作弊客户端的模块列表，显示已启用模块名称。
- **AudioVisualizer（音频可视化）：** 显示音频波形或其他可视化效果。
- **BetterLyrics（更好的歌词）：** Hook 网易云音乐并实时显示歌词。
- **Scoreboard（计分板）：** 类似 Minecraft 的计分板。
- **TimeDisplay（时间显示）：** 显示当前时间。
- **MotionBlur（动态模糊）：** 模拟屏幕动态模糊效果。
- **Clock（时钟）：** 可自定义的时钟。
- **Island（灵动岛）：** 简单的动态岛，可同步通知、任务进度和音乐歌词。
- **FancyText（花式文本）：** 以炫酷方式显示文本，类似 PV 效果。
- **Stuck（卡顿）：** 模拟卡顿效果。
- **Gamma（伽马覆写）：** 提高屏幕亮度。
- **Vape（电子烟）：** 在模块列表顶部绘制 VAPE 标志。
- **AntiRickroll（反诈骗）：** 防止被 Rickroll。
- **TcpMonitor（TCP监控）：** 监控 TCP 连接并进行通知。
- **TcpKiller（TCP终止）：** 终止 TCP 连接。
- **LiveStream（直播间）：** B 站直播辅助工具。
- **QuakeWarning（地震预警）：** 地震信息提醒。
- **BiliNotifier（B站通知）：** B 站通知同步。
- **BiliFans（粉丝监控）：** 监控 B 站粉丝数量并通知变化。
- **AntiMicrophone（反麦克风）：** 静音麦克风。
- **TimeManager（时间管理）：** 设置、偏移或加速系统时间。
- **Volume（音量）：** 通过鼠标滚轮调整系统 / 进程音量。
- **AudioIsolation（音频隔离）：** 调整进程音量，确保只有一个进程发声。
