## Modules Overview
All modules are classified into 10 categories: 
Combat, Window, Process, Registry, File, Control, Render, Web, Misc, Driver

Nice modules you can't miss:

- **AntiCapture:** Makes client window invisible from screenrecording software.
- **ClearPin:** Clear Windows PIN.
- **Gatekeeper:** Intercept some winlogin events, like Ctrl+Alt+Delete, Shutdown, Reboot, Logoff etc.
- **TrustedInstaller:** Get trustedinstaller privilege.
- **WindowHighlight:** Draw a rectangle on the activated window and a dark background beneath it.
- **OpacityChanger:** Allows you to adjust window opacity using mouse wheel.
- **ConsoleESP:** Monitor console changes. It can also real-time output content of other consoles.
- **AntiPopups:** Helps you get rid of popups.
- **DialogEscape:** Automatically close message boxes.
- **PrivateWindow:** Make some private windows (e.g. Edge, QQ, WeChat) excluded from screenrecording software.
- **ProcessManager:** An built-in application which can manage your processes.
- **CompileESP:** Monitor C/C++ compilation start and end.
- **SecurityESP:** Show a HUD of antimalware and malware process.
- **ProcessESP:** Show process creation and exit and notify you.
- **Injector:** Simple injection tool. It supports simple reflective injection.
- **FileManager:** An built-in application which can manage your files.
- **UpdateDisabler:** Delay Windows update automatically.
- **AutoClicker:** Helps you click.
- **MouseESP:** Draw multiple beautiful mouse effects.
- **ColorPicker:** Show color info your cursor points at.
- **Crosshair:** Draw a crosshair.
- **SnapTap:** Simple snap-tap module.
- **Keystrokes:** Show AWSD game keystrokes.
- **Keystrokes2:** Show dynamic keystrokes in one or multiple rows.
- **KeySound:** Highly customizable key sound module. You can use a config file.
- **ClipboardESP:** Show text or bitmap info in your clipboard (what you just copied).
- **AutoL:** Automatically send *** texts in some Minecraft occasions. No injection involved. It depends on the logs of the game.
- **ClipboardTweaks:** Approach your copied text. It includes high obfuscation methods.
- **Typewriter:** Helps you type specified content at a very fast speed.
- **Shell:** Client CLI, with a large command system. Check COMMANDS.txt for more information.
- **Arraylist:** Just like the module list in any Minecraft cheat. Shows the names of the enabled modules.
- **AudioVisualizer:** Show the audio waveform or other forms of visualization.
- **BetterLyrics:** Hook Netease Cloudmusic and show realtime lyrics.
- **Scoreboard:** Scoreboard just like the one in Minecraft.
- **TimeDisplay:** Show current time.
- **MotionBlur:** Simulated screen motion blur effect.
- **Clock:** A customizable clock.
- **Island:** A simple dynamic island, which can sync notification, task progress and music lyrics.
- **FancyText:** Show text in a fancy way. Similar to PV effects.
- **Stuck:** Fake stuck.
- **Gamma:** Brighten your monitor.
- **Vape:** Draw VAPE logo on the top of arraylist.
- **AntiRickroll:** No rickroll.
- **TcpMonitor:** Monitor TCP connections and notify you.
- **TcpKiller:** Terminate TCP connections.
- **LiveStream:** Bilibili livestream utilities.
- **QuakeWarning:** Notify the earthquake information.
- **BiliNotifier:** Bilibili notification sync.
- **BiliFans:** Monitor Bilibili follower amount and notify changes.
- **AntiMicrophone:** Mutes your mic.
- **TimeManager:** Set, offset or accelerate your system time.
- **Volume:** Helps you adjust system/process volume using mouse wheel.
- **AudioIsolation:** Adjust process volumes to promise only one process can make sounds.
